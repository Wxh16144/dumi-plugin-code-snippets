# dumi-plugin-code-snippets

> 从现有代码中导入片段

## Usage

### Install

```bash
npm install dumi-plugin-code-snippets --save-dev
```

### Apply

```ts
// .dumirc.ts
export default {
  plugins: [
    ['dumi-plugin-code-snippets', {/* options */}]
  ],
};
```
