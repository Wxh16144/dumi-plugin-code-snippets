# dumi-plugin-code-snippets

> 从现有文件中导入代码片段

## Usage

### Install

```bash
npm install dumi-plugin-code-snippets --save-dev
```

### Apply

<!-- #region enable -->

```ts
// .dumirc.ts
export default {
  plugins: ['dumi-plugin-code-snippets'],
};
```

<!-- #endregion enable -->
